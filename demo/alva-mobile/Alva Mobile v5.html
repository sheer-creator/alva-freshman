Mobile v5 · Plan C MVP

移动端只服务一个 Alva Agent Channel，但把新用户承接做深。

v5 从 v4 里删掉频道列表、群聊镜像、Personal Topic Channel 和 Skill tab。移动端只展示同一个 Agent Channel 的不同入口状态：通用新用户、KOL campaign、新手第一次任务、以及可读的 Tasks / Artifacts / Memory 骨架。

Minimum mobile surface
首页即 Alva Agent Channel，不出现 channel switcher。 Chat / Tasks / Artifacts / Memory 四个 tab 保留，SkillHub 走全局导航。 Campaign preset 只改变 onboarding recipe，不创建新 Channel。 首次交互必须能在 1 tap 内变成 task 或 automation。
