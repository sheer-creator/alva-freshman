Alva · Mobile concept · v5

单频道 MVP · 唯一 Alva Agent 频道，渠道决定开场，不决定结构

在 v4 基础上做最小 MVP 收敛：去掉频道列表、群聊镜像、Playbook 详情等扩展场景，移动端只剩一个 Alva Agent 频道——Chat / Memory / Tasks / Artifacts 只读骨架保持不变，对话仍外置 Telegram。新增「渠道化 onboarding」主线：通用注册看到欢迎语 + 全局 Skill 样例；KOL / portfolio 渠道进来的新用户，同一个频道被预置「样例推送（标 Sample）+ setup widget + 待激活 Automation + 带来源的 Memory」，填参后 Sample 变 Live，瞬间 get「Alva 是什么、我能干什么」并完成第一次交互。Skill 全局共享（Skill Hub），没有频道级 Skill 配置。

Alva 原生（teal）— 唯一频道 / 浏览 Telegram — 对话 / build 入口 Preset · Sample（amber）— campaign 预置，激活前
01

唯一频道即首屏 · 没有列表，没有切换

打开 app 直接落在 Alva Agent 频道——它是唯一的频道，所以不需要频道列表页，频道名旁也没有切换器。抽屉只承载全局入口（Explore / Portfolio / Skill Hub）+ 这一个频道。

02

渠道化 Onboarding · 同一频道，不同开场剧本

preset = 样例推送（Sample）+ setup widget + 待激活 Automation + 带来源的 Memory。KOL campaign 和 Portfolio Digest 渠道进来的新用户看到完全不同的预置内容，但频道骨架一字不差。填参后跳 Telegram 完成 build，Sample 变 Live。

03

预置骨架 · Memory 与 Artifacts 可见、可溯源

campaign 写入的一切都摆在明面上：Memory 带来源标记、可删；Artifacts 里 preset automation 以 Sample 进场、激活后变 Live。没有暗箱画像，没有伪装成用户历史的内容。
